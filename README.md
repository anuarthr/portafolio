# Portafolio Personal · Anuarth Rincón

Portafolio personal desarrollado como SPA en **React 18 + Vite + Tailwind CSS**, con animaciones en **Framer Motion**, soporte bilingüe ES/EN y despliegue en Vercel.

🌐 **[anuarthr.vercel.app](https://anuarthr.vercel.app)**

## Stack

- React 18 + Vite 5
- Tailwind CSS 3
- Framer Motion
- Formspree (formulario de contacto)
- react-icons

## Características

- Diseño responsive y modo oscuro
- Toggle de idioma Español / English
- Carrusel de imágenes con lightbox en los proyectos
- Sección de experiencia, skills y proyectos académicos
- Formulario de contacto funcional (Formspree)
- Descarga de CV según el idioma activo
